# Micro Linux Cleaner
 Update/Upgrade and clean your linux pc with this tiny python script.
 
#### Requirments 
   * Python3
```
$ sudo apt-get update
$ sudo apt-get install python3
```

## Setup Micro Linux Cleaner
```
wget https://github.com/yellucs/micro-linux-cleaner/blob/master/cleanUp.py

python3 cleanup.py 
```
